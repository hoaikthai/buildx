Component Structure

Each component has one job and is independently testable

Layout Components

Navbar Fixed top bar — logo, section links, VI/EN switcher
SideNav Numbered dot nav (1–8), tracks active section via IntersectionObserver
MobileMenu Fullscreen overlay, triggered by hamburger button
Footer Contact info, social links, tax ID, address
PageLoader Entry loading animation (matches original)

Section Components

HeroSection Full-viewport bg image, overlay, headline, CTA
AboutSection Vision/mission text, philosophy, imagery
BimSection Feature list with image slider (Embla Carousel)
DesignSection 4 service cards — structural, interior, exterior, arch
ConstructionSection Service overview with image and description
PricingSection Embla Carousel — draggable pricing cards
NewsSection Card grid — project case studies
ContactSection Map embed, phone, email, social icons

SectionTitle

LanguageSwitcher
